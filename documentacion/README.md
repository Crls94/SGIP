# Documentación Académica y Evidencias

Esta carpeta contiene documentos de soporte académico, evidencias de pruebas, datasets de preproducción, informes y presentaciones. La documentación pública del sitio MkDocs se mantiene en `docs/`.

## Estructura

| Carpeta | Contenido |
|---|---|
| `casos-prueba/` | Matrices, comandos y evidencias académicas de pruebas RF/RNF. |
| `datasets/` | Documentación de datasets de preproducción usados para Supabase/demo. |
| `informes/` | Informes académicos, análisis técnico y documentos Word/HTML generados. |
| `presentaciones/` | Presentaciones y scripts de generación asociados. |
| `referencias/` | Material de referencia externo usado para estructurar informes. |

## Notas

- Los archivos `.docx` y `.pptx` generados localmente están ignorados para evitar ruido en Git, salvo documentos ya versionados previamente.
- Los dumps de base de datos no deben versionarse; usar respaldos locales o almacenamiento seguro externo.
- Para documentación navegable del proyecto, usar `mkdocs.yml` y la carpeta `docs/`.
